Service UI
Thor's main service ui is a graphical representation of the tree, and although changes to the tree can be done through code, it is more likely that you will do most of them through the UI.

The Main View

Adding or modifying an Item

Whether you are modifiying or adding an item you will be presented with the same dialog

If you select XML, you will be asked to select a compliant XML file, see Adding bulk data and branches in bulk
On completing your changes and clicking on OK, you will be asked to confirm your choices
